Summary

Chart.js is a free javascript library that can be used to create simple charts and graphs. It supports six different types of charts: bar, line, pie, radar, doughnut, and polar. Chart.js uses the HTML5 canvas and supports animations.

Pros

  • Free and open source
  • Easy to learn and use
  • Excellent documentation at https://www.chartjs.org/docs/latest/
  • Responsive to different size windows
  • Low impact on page loading

Cons

  • Not much support for advanced features
